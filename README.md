# Python Class Projects

> A collection of Python exercises and OOP concepts from coursework at Auburn University — covers core Python programming, object-oriented design, file handling, and more.

## Topics Covered

- **OOP Concepts** — Classes, Abstraction, Encapsulation, Inheritance, Polymorphism
- **Access Control** — Private attributes and access modifiers
- **File Handling** — Reading and writing files in Python
- **Data Structures** — Lists, dictionaries, tuples
- **Functions & Modules** — Reusable Python code patterns

## File Structure

| File | Description |
|------|-------------|
| `AK_1.py` | Initial Python exercises |
| `AK_2.py` | Intermediate Python practice |
| `AK_3.py` | Advanced Python concepts |
| `AK_file.py` | OOP concepts with file handling |
| `Abstraction.py` | Abstraction and access control |
| `# private_access.py` | Private attribute examples |

## Getting Started

### Prerequisites

- Python 3.8+

### Run any file

```bash
python AK_1.py
```
